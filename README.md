## 🌌 專案願景 (Project Vision)
`DualBear_Sentinel_Alpha` 旨在解決資訊超載導致的「資訊勞動束縛」。透過 **Jade (理性精算)** 與 **Opal (戰略遠見)** 的雙重邏輯，本系統自動從台股社群（PTT）與官方新聞（Anue）中提煉群眾情緒，並轉化為可量化的交易信號，協助使用者在混沌的市場中覺醒，實現時間與財務的自由。

## 🛠️ 核心戰略引擎 (Core Engine)
- **多因子情緒加權 (Multi-factor Weighting)**：整合社群散戶情緒 (40%)、官方新聞 (20%) 與法規總經 (30%)。
- **酸民語意引擎 (Sarcasm Detection)**：專為 PTT/Dcard 設計，能識破「反串」與「黑話」，捕捉真實的恐慌與貪婪。
- **自動化戰略對齊**：每日定時同步股價與情緒，計算「情緒背離」信號。
- **火力控制系統 (Position Sizing)**：根據情緒總分自動建議 0% - 100% 的倉位管理。

## 🏗️ 系統架構 (Architecture)


- **Data Lake**: Scrapy & Playwright (PTT, Anue, Yahoo Finance)
- **AI Brain**: OpenAI GPT-4o-mini (Sentiment & Sarcasm Analysis)
- **Storage**: SQLite (Strategic Metadata Vault)
- **Dashboard**: Streamlit (Visual Multi-factor Analytics)
- **Action**: GitHub Actions + LINE Notify (24/7 Monitoring)

## 🚀 快速啟動 (Quick Start)
1. **設定金鑰**：於 GitHub Secrets 中配置 `OPENAI_API_KEY` 與 `LINE_NOTIFY_TOKEN`。
2. **初始化**：執行 `python initialize_vault.py` 建立數位金庫。
3. **雲端掛載**：Push 程式碼至 GitHub 觸發 Actions。
